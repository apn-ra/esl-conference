# Command Vocabulary

## Stable Commands

| Command | Status | Notes |
| --- | --- | --- |
| `conference <name> list` | stable 0.1.0 | Generated by `ConferenceListMembersCommand`. |

Command builders only generate command text. They do not execute commands and do not prepend `api` or `bgapi`.

## Blockers

- `conference_command_argument_unsafe`
- `conference_command_rejected`
- `conference_vocabulary_unproven`

## Unsupported Commands

`json_list`, `bgdial`, member administration families, recording commands, media playback commands, transfer commands, and relate commands are unsupported in the stable 0.1.0 API.
